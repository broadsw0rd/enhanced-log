Beauty Log

Make your console.log more sexy

Why?

How often you doing things such this when debug your code?

console.log('================================')
console.log(super.iportant.debug.info)
console.log('================================')

Now you have very clever way to avoid this with beauty-log

About

beauty-log have several methods which very similar to CSS classes and you can combine them as you like. Just open your Dev Tools to see it in action

Method Result
log#large message
log#huge message
log#small message
log#info message
log#success message
log#warning message
log#danger message
log#underline message
log#overline message
log#linethrough message
log#capitalize message
log#uppercase message
log#lowercase message
log#bold message
log#italic message

Installation

via Bower

$ bower install beauty-log
<script src="bower_components/beauty-log/dist/log.min.js"></script>

via npm

$ npm install beauty-log
var log = require('beauty-log')

And now you have full power of log function

Support

Each browser which support styling console output with CSS
